# Button

A pressable button with animated press feedback, three visual variants, three size presets, loading state, icon slots, and full theme integration.

## Usage

```tsx
import { Button } from "@gooey/core";

<Button onPress={() => save()}>Save changes</Button>
```

## Sizes

```tsx
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>  {/* default */}
<Button size="lg">Large</Button>
```

## Loading state

The button shows a spinner and disables interaction while `loading` is true:

```tsx
const [saving, setSaving] = useState(false);

<Button
  loading={saving}
  onPress={async () => {
    setSaving(true);
    await save();
    setSaving(false);
  }}
>
  Save changes
</Button>
```

## Icons

```tsx
import { PlusIcon, ChevronRightIcon } from "your-icon-library";

<Button leftIcon={<PlusIcon size={16} />}>Add item</Button>
<Button rightIcon={<ChevronRightIcon size={16} />}>Next</Button>
```

## Custom color

Override the button's primary color independently of the theme:

```tsx
<Button color="#7C3AED">Brand action</Button>
<Button color="#10B981" variant="solid">Success action</Button>
```

<Callout type="info">
  `color` overrides the background for `solid` buttons and the border/text color for `outline` and `ghost` buttons.
</Callout>

## Disabled

```tsx
<Button disabled>Unavailable</Button>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `children` | `ReactNode` | — | Button label |
| `variant` | `"solid" \| "outline" \| "ghost"` | `"solid"` | Visual style |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size preset |
| `color` | `string` | — | Override the accent color |
| `loading` | `boolean` | `false` | Show spinner, disable interaction |
| `disabled` | `boolean` | `false` | Disable interaction |
| `leftIcon` | `ReactNode` | — | Icon before the label |
| `rightIcon` | `ReactNode` | — | Icon after the label |
| `onPress` | `() => void` | — | Press handler |
| `style` | `StyleProp<ViewStyle>` | — | Container style override |
| `accessibilityLabel` | `string` | — | Screen reader label |
