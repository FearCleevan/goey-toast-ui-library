# @gooey/core

The foundation package. Includes the morphing blob toast system, a full theming engine, and 20+ production-ready React Native components.

## Install

<Tabs items={["Expo", "pnpm", "npm", "yarn"]}>
  <Tabs.Tab>
  ```bash
  npx expo install @gooey/core react-native-reanimated react-native-gesture-handler react-native-svg react-native-safe-area-context
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  pnpm add @gooey/core
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  npm install @gooey/core
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  yarn add @gooey/core
  ```
  </Tabs.Tab>
</Tabs>

<Callout type="info">
  Peer dependencies: `react-native-reanimated ≥ 3`, `react-native-gesture-handler ≥ 2`, `react-native-svg ≥ 15`, `react-native-safe-area-context ≥ 4`.
</Callout>
