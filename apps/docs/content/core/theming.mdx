# Theming

All goey-toast components read from a shared `ITheme` object. The theme provides colors, spacing, radius, typography, motion, and shadow tokens.

## Built-in themes

```tsx
import { lightTheme, darkTheme } from "@gooey/core";

<ThemeProvider theme={darkTheme}>
  {/* ... */}
</ThemeProvider>
```

## Create a custom theme

`createTheme` deep-merges your overrides with `lightTheme`:

```tsx
import { createTheme, ThemeProvider } from "@gooey/core";

const brandTheme = createTheme({
  colors: {
    primary: "#6C47FF",
    background: "#0D0D0D",
    foreground: "#F5F5F5",
  },
});

<ThemeProvider theme={brandTheme}>
  {/* ... */}
</ThemeProvider>
```

## useTheme

Access design tokens inside any component:

```tsx
import { useTheme } from "@gooey/core";

function MyCard() {
  const theme = useTheme();

  return (
    <View style={{
      backgroundColor: theme.colors.card,
      borderRadius: theme.radius.xl,
      padding: theme.spacing[4],
    }}>
      {/* ... */}
    </View>
  );
}
```

## Token reference

### Colors

| Token | Light | Dark |
|---|---|---|
| `colors.background` | `#FFFFFF` | `#09090B` |
| `colors.foreground` | `#09090B` | `#FAFAFA` |
| `colors.card` | `#FFFFFF` | `#18181B` |
| `colors.border` | `#E4E4E7` | `#27272A` |
| `colors.muted` | `#F4F4F5` | `#27272A` |
| `colors.mutedForeground` | `#71717A` | `#A1A1AA` |

### Spacing

Spacing uses a 4px base scale — `theme.spacing[1]` = `4px`, `theme.spacing[4]` = `16px`, `theme.spacing[8]` = `32px`.

### Radius

```ts
theme.radius.sm   // 4
theme.radius.md   // 8
theme.radius.lg   // 12
theme.radius.xl   // 16
theme.radius["2xl"] // 20
theme.radius["3xl"] // 24
theme.radius.full // 9999
```

### Palette

The full color palette (`palette`) is exported as a flat object:

```ts
import { palette } from "@gooey/core";

palette.blue600   // "#2563EB"
palette.slate200  // "#E2E8F0"
palette.green500  // "#22C55E"
```

<Callout type="info">
  The palette is a **flat** object — use `palette.blue600`, not `palette.blue[600]`.
</Callout>
