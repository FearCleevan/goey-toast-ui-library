# Theming

Every gooey-toast component reads from a shared `ITheme` object. The theme provides tokens for colors, spacing, radius, typography, motion, and shadows. You can use the built-in light or dark theme, or create a fully custom one.

---

## Create a custom theme

`createTheme` deep-merges your overrides into `lightTheme`. Only the keys you provide are changed — everything else inherits from the base.

```tsx
import { ThemeProvider, createTheme } from "@gooey/core";

const brandTheme = createTheme({
  colors: {
    primary: "#7C3AED",
    background: "#09090B",
    foreground: "#FAFAFA",
    card: "#18181B",
    border: "#27272A",
  },
  radius: {
    md: 12,
    lg: 16,
  },
});

<ThemeProvider theme={brandTheme}>
  {/* ... */}
</ThemeProvider>
```

<Callout type="info">
  `createTheme` always starts from `lightTheme`. To extend `darkTheme`, spread it first: `createTheme({ ...darkTheme, colors: { ...darkTheme.colors, primary: "#7C3AED" } })`.
</Callout>

## useTheme

Access design tokens inside any component wrapped in `ThemeProvider`:

```tsx
import { useTheme } from "@gooey/core";
import { View, StyleSheet } from "react-native";

function MyCard() {
  const theme = useTheme();

  return (
    <View
      style={{
        backgroundColor: theme.colors.card,
        borderColor: theme.colors.border,
        borderWidth: 1,
        borderRadius: theme.radius.xl,
        padding: theme.spacing[4],
      }}
    >
      {/* ... */}
    </View>
  );
}
```

## Token reference

### Colors

| Token | Light | Dark |
|---|---|---|
| `colors.background` | `#FFFFFF` | `#09090B` |
| `colors.foreground` | `#09090B` | `#FAFAFA` |
| `colors.card` | `#FFFFFF` | `#18181B` |
| `colors.border` | `#E4E4E7` | `#27272A` |
| `colors.muted` | `#F4F4F5` | `#27272A` |
| `colors.mutedForeground` | `#71717A` | `#A1A1AA` |
| `colors.primary` | `#7C3AED` | `#7C3AED` |

### Spacing

Spacing uses a **4 px base scale**:

| Token | Value |
|---|---|
| `spacing[1]` | 4 px |
| `spacing[2]` | 8 px |
| `spacing[3]` | 12 px |
| `spacing[4]` | 16 px |
| `spacing[6]` | 24 px |
| `spacing[8]` | 32 px |
| `spacing[12]` | 48 px |
| `spacing[16]` | 64 px |

### Radius

| Token | Value |
|---|---|
| `radius.sm` | 4 |
| `radius.md` | 8 |
| `radius.lg` | 12 |
| `radius.xl` | 16 |
| `radius["2xl"]` | 20 |
| `radius["3xl"]` | 24 |
| `radius.full` | 9999 |

### Palette

The full color palette is exported as a **flat** object — use `palette.blue600`, not `palette.blue[600]`:

```ts
import { palette } from "@gooey/core";

palette.blue600    // "#2563EB"
palette.violet500  // "#8B5CF6"
palette.green500   // "#22C55E"
palette.red500     // "#EF4444"
palette.amber400   // "#FBBF24"
palette.slate200   // "#E2E8F0"
```

### Motion

```ts
theme.motion.duration.fast    // 150 ms
theme.motion.duration.normal  // 300 ms
theme.motion.duration.slow    // 500 ms

theme.motion.easing.spring    // Reanimated spring config
theme.motion.easing.ease      // cubic-bezier easing
```
