# @gooey/blocks

Pre-built screen compositions for the most common mobile app flows. Every block is built from `@gooey/core` primitives and fires `gooeyToast` feedback automatically.

## Install

<Tabs items={["pnpm", "npm", "yarn", "bun"]}>
  <Tabs.Tab>
  ```bash
  pnpm add @gooey/blocks @gooey/core
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  npm install @gooey/blocks @gooey/core
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  yarn add @gooey/blocks @gooey/core
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  bun add @gooey/blocks @gooey/core
  ```
  </Tabs.Tab>
</Tabs>

<Callout type="info">
  All blocks peer-depend on `@gooey/core`. Make sure it is installed and `ThemeProvider` wraps your app before using any block.
</Callout>

## Suppressing toasts

Every block that fires toasts accepts a `disableToast` prop. Set it to suppress all automatic `gooeyToast` calls — useful during testing or when you want full control over feedback:

```tsx
<SignInBlock onSignIn={handleSignIn} disableToast />
<ProductCard onAddToCart={addToCart} disableToast />
```
