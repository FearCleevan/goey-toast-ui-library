# ActivityFeed

A scrollable list of timestamped activity rows with type-colored dot indicators, optional badges, and toast-on-press feedback.

## Usage

```tsx
import { ActivityFeed } from "@gooey/blocks";

const items = [
  {
    id: "1",
    label: "Invoice #1042 paid",
    timestamp: "2 min ago",
    type: "success",
    badge: "Paid",
  },
  {
    id: "2",
    label: "Deployment failed",
    timestamp: "18 min ago",
    type: "error",
    badge: "Failed",
  },
  {
    id: "3",
    label: "New user signed up",
    timestamp: "1 hr ago",
    type: "info",
  },
];

<ActivityFeed
  items={items}
  onItemPress={(item) => router.push(`/activity/${item.id}`)}
/>
```

## Toast behavior

When `onItemPress` is provided, pressing an item fires:
```
gooeyToast(item.label, { type: item.type, description: item.timestamp })
```

## Props

| Prop | Type | Description |
|---|---|---|
| `items` | `IActivityItem[]` | Feed data |
| `onItemPress` | `(item) => void` | Item press handler |
| `disableToast` | `boolean` | Suppress automatic toast on press |
| `style` | `StyleProp<ViewStyle>` | Container style |

## IActivityItem

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique key |
| `label` | `string` | Activity description |
| `timestamp` | `string` | Human-readable time |
| `type` | `"success" \| "error" \| "warning" \| "info"` | Colors the dot indicator |
| `badge` | `string` | Optional badge label |
