# Getting Started

Get gooey-toast running in your React Native or Expo project in under five minutes.

## Requirements

| Dependency | Minimum version |
|---|---|
| React Native | 0.74 |
| React | 18 |
| Expo SDK _(optional)_ | 51 |
| react-native-reanimated | 3.0 |
| react-native-gesture-handler | 2.0 |
| react-native-svg | 15.0 |
| react-native-safe-area-context | 4.0 |

### Configure the Babel plugin

Reanimated requires its Babel plugin. Add it to `babel.config.js`:

<Tabs items={["Expo", "React Native CLI"]}>
  <Tabs.Tab>
  ```js filename="babel.config.js"
  module.exports = function (api) {
    api.cache(true);
    return {
      presets: ["babel-preset-expo"],
      plugins: ["react-native-reanimated/plugin"],
    };
  };
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```js filename="babel.config.js"
  module.exports = {
    presets: ["module:@react-native/babel-preset"],
    plugins: ["react-native-reanimated/plugin"],
  };
  ```
  </Tabs.Tab>
</Tabs>

<Callout type="warning">
  The Reanimated plugin **must be listed last** in the `plugins` array. After adding it, clear Metro cache: `npx expo start --clear` or `npx react-native start --reset-cache`.
</Callout>

### Wrap your root layout

Place `ThemeProvider` and `GooeyToaster` once at the root of your app.

<Tabs items={["Expo Router", "React Navigation", "Bare RN"]}>
  <Tabs.Tab>
  ```tsx filename="app/_layout.tsx"
  import { ThemeProvider, GooeyToaster } from "@gooey/core";
  import { GestureHandlerRootView } from "react-native-gesture-handler";
  import { SafeAreaProvider } from "react-native-safe-area-context";
  import { Stack } from "expo-router";

  export default function RootLayout() {
    return (
      <GestureHandlerRootView style={{ flex: 1 }}>
        <SafeAreaProvider>
          <ThemeProvider>
            <Stack />
            <GooeyToaster />
          </ThemeProvider>
        </SafeAreaProvider>
      </GestureHandlerRootView>
    );
  }
  ```
  </Tabs.Tab>
  <Tabs.Tab>

### Fire your first toast

Call `gooeyToast` from anywhere — no hooks, no context required:

```tsx
import { gooeyToast } from "@gooey/core";

// Variants
gooeyToast.success("Saved!", { description: "Your changes have been saved." });
gooeyToast.error("Upload failed", { description: "Check your connection." });
gooeyToast.warning("Low storage", { duration: 6000 });
gooeyToast.info("Update available");

// With an action button
gooeyToast("File deleted", {
  description: "invoice.pdf",
  action: { label: "Undo", onPress: () => restoreFile() },
  duration: 6000,
});

// Promise toast
gooeyToast.promise(uploadFile(), {
  loading: "Uploading…",
  success: (data) => `Uploaded ${data.name}`,
  error: (err) => err.message,
});
```
