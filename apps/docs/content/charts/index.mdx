# @gooey/charts

Themed chart components for React Native, built on [Victory Native](https://commerce.nearform.com/open-source/victory-native/).

## Install

```bash
npx expo install @gooey/charts victory-native react-native-svg
```

<Callout type="info">
  `@gooey/charts` uses the **SVG-based** Victory Native API (`victory-native@^36`). It does not require Skia or the new XL API.
</Callout>

## Available charts

| Component | Description |
|---|---|
| `BarChart` | Vertical bar chart with optional tooltip |
| `LineChart` | Line chart with scatter points |
| `AreaChart` | Filled area chart |
| `PieChart` | Slice-based pie chart |
| `DonutChart` | Donut variant with center label |

## Theming

All charts read from `@gooey/core`'s theme automatically — no extra config needed. Colors come from `theme.colors.toast.*` icon values (info, success, warning, error) plus extended palette entries.
