# AreaChart

A filled area chart — useful for showing cumulative totals or trends over time.

## Usage

```tsx
import { AreaChart } from "@gooey/charts";

const data = [
  { x: "Jan", y: 4000 },
  { x: "Feb", y: 3200 },
  { x: "Mar", y: 5600 },
  { x: "Apr", y: 4800 },
];

<AreaChart data={data} height={220} />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Array<{ x: string \| number; y: number }>` | — | Chart data |
| `height` | `number` | `240` | Chart height |
| `color` | `string` | theme blue | Area fill and line color |
| `fillOpacity` | `number` | `0.2` | Area fill opacity |
| `onPress` | `(datum) => void` | — | Datum press callback |
| `toastConfig` | `IChartToastConfig` | — | Toast on hover |
| `style` | `StyleProp<ViewStyle>` | — | Container style |
