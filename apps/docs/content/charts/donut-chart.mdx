# DonutChart

A donut chart with an optional center label — great for showing a single metric with breakdown.

## Usage

```tsx
import { DonutChart } from "@gooey/charts";

const data = [
  { x: "Completed", y: 68 },
  { x: "In Progress", y: 22 },
  { x: "Pending", y: 10 },
];

<DonutChart
  data={data}
  height={280}
  centerLabel="68%"
  centerSubLabel="Done"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Array<{ x: string; y: number }>` | — | Slice data |
| `height` | `number` | `280` | Chart height |
| `innerRadius` | `number` | `0.55` | Hole size (0–1 as fraction of radius) |
| `centerLabel` | `string` | — | Large text in the center hole |
| `centerSubLabel` | `string` | — | Smaller text below center label |
| `colors` | `string[]` | theme palette | Per-slice colors |
| `onSlicePress` | `(datum) => void` | — | Slice press callback |
| `toastConfig` | `IChartToastConfig` | — | Toast on slice press |
| `style` | `StyleProp<ViewStyle>` | — | Container style |
