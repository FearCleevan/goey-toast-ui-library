# PieChart

A slice-based pie chart with auto-colored segments from the Gooey palette.

## Usage

```tsx
import { PieChart } from "@gooey/charts";

const data = [
  { x: "Food", y: 35 },
  { x: "Transport", y: 20 },
  { x: "Housing", y: 30 },
  { x: "Other", y: 15 },
];

<PieChart data={data} height={300} />
```

## With press callback

```tsx
<PieChart
  data={data}
  height={300}
  onSlicePress={(datum) => console.log("Slice:", datum.x)}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Array<{ x: string; y: number }>` | — | Slice data |
| `height` | `number` | `300` | Chart height |
| `colors` | `string[]` | theme palette | Per-slice colors |
| `onSlicePress` | `(datum) => void` | — | Slice press callback |
| `toastConfig` | `IChartToastConfig` | — | Toast on slice press |
| `style` | `StyleProp<ViewStyle>` | — | Container style |
