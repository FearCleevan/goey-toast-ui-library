# ProductCard

An e-commerce product card with image, name, price, optional original price (strikethrough), optional badge, and an "Add to cart" button with built-in toast feedback.

## Usage

```tsx
import { ProductCard } from "@gooey/blocks";

<ProductCard
  name="Wireless Noise-Cancelling Headphones"
  price="$149.99"
  originalPrice="$199.99"
  imageUri="https://example.com/headphones.jpg"
  badge="Sale"
  onAddToCart={() => cart.add(productId)}
  onPress={() => router.push(`/product/${productId}`)}
/>
```

## Toast on add to cart

Pressing "Add to cart" fires: `gooeyToast.success("Added to cart", { description: name })`

Set `disableToast` to suppress it.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `name` | `string` | — | Product name |
| `price` | `string` | — | Current price (formatted) |
| `originalPrice` | `string` | — | Original price, shown with strikethrough |
| `imageUri` | `string` | — | Product image URI |
| `badge` | `string` | — | Badge label (e.g. "Sale", "New") |
| `onAddToCart` | `() => void` | — | Renders "Add to cart" button if provided |
| `onPress` | `() => void` | — | Card press handler |
| `disableToast` | `boolean` | `false` | Suppress cart toast |
| `style` | `StyleProp<ViewStyle>` | — | Container style |
