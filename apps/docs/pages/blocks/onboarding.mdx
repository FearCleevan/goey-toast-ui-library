# OnboardingFlow

A paginated onboarding flow with animated pagination dots, illustration slots, per-slide accent colors, and an optional skip button.

## Usage

```tsx
import { OnboardingFlow } from "@gooey/blocks";
import { Text } from "@gooey/core";

const slides = [
  {
    id: "welcome",
    title: "Welcome to MyApp",
    description: "The best way to manage your tasks.",
    illustration: <Text size="4xl">👋</Text>,
    accentColor: "#6C47FF",
  },
  {
    id: "collaborate",
    title: "Collaborate",
    description: "Share projects and work together in real time.",
    illustration: <Text size="4xl">🤝</Text>,
    accentColor: "#0EA5E9",
  },
  {
    id: "ship",
    title: "Ship faster",
    description: "Automate workflows and get more done.",
    illustration: <Text size="4xl">🚀</Text>,
    accentColor: "#22C55E",
  },
];

<OnboardingFlow
  slides={slides}
  onComplete={() => router.replace("/home")}
  onSkip={() => router.replace("/home")}
  ctaLabel="Let's go"
/>
```

## Toast behavior

On completing the last slide: `gooeyToast.success("You're all set!", { description: "Let's get started." })`

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `slides` | `IOnboardingSlideData[]` | — | Slide definitions |
| `onComplete` | `() => void` | — | Called when the last slide CTA is pressed |
| `onSkip` | `() => void` | — | Renders a "Skip" button if provided |
| `ctaLabel` | `string` | `"Get started"` | Final slide CTA button label |
| `disableToast` | `boolean` | `false` | Suppress completion toast |

## IOnboardingSlideData

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique key |
| `title` | `string` | Slide heading |
| `description` | `string` | Slide body text |
| `illustration` | `ReactNode` | Optional illustration (centered) |
| `accentColor` | `string` | Overrides the accent dot and CTA button color |
