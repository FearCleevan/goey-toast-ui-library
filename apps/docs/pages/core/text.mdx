# Text

A themed `Text` component that maps semantic size/weight tokens to React Native styles.

## Usage

```tsx
import { Text } from "@gooey/core";

<Text size="lg" weight="bold" color={theme.colors.foreground}>
  Hello world
</Text>
```

## Sizes

```tsx
<Text size="xs">Extra small</Text>
<Text size="sm">Small</Text>
<Text size="md">Medium (default)</Text>
<Text size="lg">Large</Text>
<Text size="xl">Extra large</Text>
<Text size="2xl">2X large</Text>
<Text size="3xl">3X large</Text>
<Text size="4xl">4X large</Text>
```

## Weights

```tsx
<Text weight="normal">Normal</Text>
<Text weight="medium">Medium</Text>
<Text weight="semibold">Semibold</Text>
<Text weight="bold">Bold</Text>
```

## Alignment

```tsx
<Text align="left">Left</Text>
<Text align="center">Center</Text>
<Text align="right">Right</Text>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl"` | `"md"` | Font size preset |
| `weight` | `"normal" \| "medium" \| "semibold" \| "bold"` | `"normal"` | Font weight |
| `color` | `string` | `theme.colors.foreground` | Text color |
| `align` | `"left" \| "center" \| "right"` | `"left"` | Text alignment |
| `numberOfLines` | `number` | — | Line clamp (passed to RN Text) |
| `style` | `StyleProp<TextStyle>` | — | Additional styles |
