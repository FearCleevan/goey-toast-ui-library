# @gooey/core

The foundation package. Includes the toast system, theming engine, and 20+ React Native components.

## Install

```bash
npx expo install @gooey/core react-native-reanimated
```

## What's included

| Category | Exports |
|---|---|
| **Toast** | `GooeyToaster`, `gooeyToast`, `removeToast`, `dismissAll` |
| **Theme** | `ThemeProvider`, `useTheme`, `createTheme`, `lightTheme`, `darkTheme` |
| **Layout** | `Box`, `HStack`, `VStack`, `Stack`, `Divider` |
| **Typography** | `Text` |
| **Inputs** | `Input`, `Button`, `IconButton`, `Switch`, `Checkbox` |
| **Display** | `Avatar`, `Badge`, `Card`, `Chip`, `Alert` |
| **Overlays** | `Modal`, `BottomSheet` |
| **Feedback** | `Skeleton` |
| **Motion** | `motion` config object |
| **Tokens** | `palette`, `radius`, `spacing`, `typography` |
