# Layout Primitives

`HStack`, `VStack`, `Box`, and `Divider` are the building blocks for all layouts in goey-toast.

## VStack

Vertical stack (column direction) with a `gap` prop:

```tsx
import { VStack, Text } from "@gooey/core";

<VStack gap={12}>
  <Text>First item</Text>
  <Text>Second item</Text>
</VStack>
```

## HStack

Horizontal stack (row direction):

```tsx
import { HStack, Text } from "@gooey/core";

<HStack gap={8} align="center" justify="space-between">
  <Text>Left</Text>
  <Text>Right</Text>
</HStack>
```

## Stack props

Both `VStack` and `HStack` share the same props:

| Prop | Type | Description |
|---|---|---|
| `gap` | `number` | Space between children |
| `align` | `"flex-start" \| "center" \| "flex-end" \| "stretch"` | Cross-axis alignment |
| `justify` | `"flex-start" \| "center" \| "flex-end" \| "space-between" \| "space-around" \| "space-evenly"` | Main-axis alignment |
| `flex` | `number` | `flex` style shorthand |
| `wrap` | `boolean` | Enable `flexWrap: "wrap"` |
| `style` | `StyleProp<ViewStyle>` | Additional styles |

## Box

A low-level layout primitive with shorthand style props:

```tsx
import { Box } from "@gooey/core";

<Box p={4} radius="lg" bg={theme.colors.card}>
  {/* content */}
</Box>
```

| Prop | Maps to |
|---|---|
| `p` / `px` / `py` / `pt` / `pb` / `pl` / `pr` | padding* |
| `m` / `mx` / `my` / `mt` / `mb` / `ml` / `mr` | margin* |
| `bg` | `backgroundColor` |
| `radius` | `borderRadius` key from `theme.radius` |
| `shadow` | shadow token key |
| `borderColor` | `borderColor` |
| `borderWidth` | `borderWidth` |
| `flex` | `flex` |
| `row` | `flexDirection: "row"` |
| `centered` | `alignItems: "center", justifyContent: "center"` |

## Divider

A horizontal or vertical separator line:

```tsx
import { Divider } from "@gooey/core";

<Divider />
<Divider orientation="vertical" />
<Divider color={theme.colors.border} thickness={2} />
```

| Prop | Type | Default |
|---|---|---|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |
| `color` | `string` | `theme.colors.border` |
| `thickness` | `number` | `1` |
