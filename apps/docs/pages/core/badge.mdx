# Badge

A small label chip for status, counts, and categories.

## Usage

```tsx
import { Badge } from "@gooey/core";

<Badge variant="success">Active</Badge>
<Badge variant="error">Failed</Badge>
<Badge variant="warning">Pending</Badge>
<Badge variant="info">New</Badge>
<Badge variant="neutral">Draft</Badge>
```

## Dot mode

Renders as a colored dot with no text — useful for presence indicators:

```tsx
<Badge dot variant="success">{null}</Badge>
```

## Sizes

```tsx
<Badge size="sm" variant="info">Small</Badge>
<Badge size="md" variant="info">Medium</Badge>
<Badge size="lg" variant="info">Large</Badge>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `children` | `ReactNode` | — | Badge label content |
| `variant` | `"success" \| "error" \| "warning" \| "info" \| "neutral"` | `"neutral"` | Color variant |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Size preset |
| `dot` | `boolean` | `false` | Render as dot (no text) |
| `style` | `StyleProp<ViewStyle>` | — | Container style override |
