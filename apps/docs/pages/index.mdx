# goey-toast

**Morphing blob toasts and a full React Native UI library** — batteries included.

goey-toast started as a delightful animated toast library for React Native / Expo and has grown into a complete UI ecosystem with components, charts, pre-built screen blocks, and a CLI installer.

## Quick install

```bash
# npm
npm install @gooey/core

# pnpm
pnpm add @gooey/core

# yarn
yarn add @gooey/core

# bun
bun add @gooey/core
```

Then wrap your app:

```tsx
import { ThemeProvider, GooeyToaster } from "@gooey/core";

export default function App() {
  return (
    <ThemeProvider>
      <YourApp />
      <GooeyToaster />
    </ThemeProvider>
  );
}
```

Trigger a toast from anywhere:

```tsx
import { gooeyToast } from "@gooey/core";

gooeyToast.success("Saved!", { description: "Your changes have been saved." });
```
