# gooey list

Shows all components available in the registry, their tier, description, and whether they've been added to your project.

## Usage

```bash
gooey list
```

## Example output

```
◆ Available components

  core
  ────────────────────────────────────────────────
  ✔ SignInBlock      Full sign-in form with toast feedback
  ✔ SignUpBlock      Full sign-up form with validation
    StatsCard        Metric card with change badge

  composed
  ────────────────────────────────────────────────
    OnboardingFlow   Paginated onboarding slides
    ProductCard      E-commerce product card
    CartItem         Cart row with quantity stepper

  ✔ = already added to this project
```
