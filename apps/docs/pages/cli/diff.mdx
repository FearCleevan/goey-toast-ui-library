# gooey diff

Compares your local copy of a component against the current version in the registry — useful after library updates.

## Usage

```bash
gooey diff <component-name>
gooey diff SignInBlock
```

## Example output

```
◆ Diff: SignInBlock
  Local:    src/components/gooey/SignInBlock.tsx
  Registry: SignInBlock@latest

  - gooeyToast.warning("Missing fields", {
  + gooeyToast.warning("Required fields missing", {
      description: "Please fill in all fields to continue.",
  -   duration: 3000,
  +   duration: 2500,
    });
```

Green lines (`+`) are additions in the registry version. Red lines (`-`) are lines in your local copy that differ or were removed.

## When to use

Run `gooey diff` after upgrading `@gooey/core` or `@gooey/blocks` to see if the underlying template has changed in ways you may want to merge into your local copy.
