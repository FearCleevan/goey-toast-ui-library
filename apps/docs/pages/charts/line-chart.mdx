# LineChart

A smooth line chart with scatter points and optional Voronoi tooltip.

## Usage

```tsx
import { LineChart } from "@gooey/charts";

const data = [
  { x: 1, y: 30 },
  { x: 2, y: 55 },
  { x: 3, y: 40 },
  { x: 4, y: 80 },
  { x: 5, y: 65 },
];

<LineChart data={data} height={220} />
```

## Multiple series

```tsx
<LineChart
  data={revenue}
  height={220}
  color="#6C47FF"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `Array<{ x: number; y: number }>` | — | Chart data points |
| `height` | `number` | `240` | Chart height |
| `color` | `string` | theme blue | Line and dot color |
| `showDots` | `boolean` | `true` | Render scatter points |
| `smooth` | `boolean` | `true` | Smooth interpolation |
| `onPress` | `(datum) => void` | — | Datum press callback |
| `toastConfig` | `IChartToastConfig` | — | Voronoi toast on hover |
| `style` | `StyleProp<ViewStyle>` | — | Container style |
